ARCHIVE MODE

FASTA EXPORT
Exports the DNA sequence in FASTA genomic format compatible with bioinformatics workflows.

HOW TO ENCODE
1. Paste plaintext into the input box.
2. Select export mode.
3. Press ENCODE.
4. Export using .DNA or .TXT.

HOW TO DECODE
RAW DNA sequences decode perfectly.
Paste only the DNA payload consisting of A/C/G/T bases, then press DECODE.
