# 🧬 Vespera Helix: Genomic Transcoder

**Vespera Helix** is a minimalist, bidirectional data-to-DNA transcoder. It bridges the gap between digital text (Base-2) and genomic sequences (Base-4), simulating the logic used in high-density synthetic DNA data storage.

---

## 🛰️ Overview
Developed by **Vespera Systems**, Helix provides a sleek interface for converting standard UTF-8 text into Adenine (A), Cytosine (C), Guanine (G), and Thymine (T) sequences.

### Core Features:
*   **Real-time Transcoding**: Convert text to DNA and DNA to text instantly.
*   **Auto-Detect Logic**: The system identifies whether the input is a biological sequence or raw text.
*   **Zero Dependencies**: A lightweight, single-file HTML5/JS solution.
*   **Vespera Biotech Interface**: Dark-mode optimized for lab and developer environments.

## 🧪 The Logic
Helix utilizes a 2-bit mapping system to represent digital data within biological bases:
- `00` → **A** (Adenine)
- `01` → **C** (Cytosine)
- `10` → **G** (Guanine)
- `11` → **T** (Thymine)
